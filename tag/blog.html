我的第一个博客！

Date Tue 25 August 2015 By Ben Category Computer. Tags blog pelican markdown

Blog Setup

之前和同学一起学unity3D, 在Iteye上面写博客写了几篇就坚持不下来了。。。这几天突然想着要搭一个自己的博客。各种google，百度终于在框架搭好了，记录一下(中间ubuntu, windows两个系统各种折腾，终于都打好了一 一+)

Linux下的搭建

选用的工具是python + pelican + github 搭建一个静态博客， 参考网站很多，主要是这个GITHUB个人博客搭建-Pelican 在Windows环境下的安装及配置以及"用Pelican和Github Pages在Linux上搭建个人博客"。

首先你得有个github帐号， 官网申请完之后，建立一个username.github.io的版本库，然后把你的博客放进这个库中，然后就可以直接登录username.github.io的网址看见你的博客了

然后选用的框架是pelican, 看到说是基于python的我就直接选了这个了，反正是第一次。。 先把这个pip的安装程序弄下来get-pip.py

sudo apt-get install python   #安装Python
python get-pip.py ...
